# Preentrega
Pagina web de comercio de ventas de zapatillas con html.con estilos css,con formulario con flexbox y formulario de contacto 
El propósito de esta página es proporcionar a los clientes una plataforma digital donde puedan:
- Explorar un amplio catálogo de zapatillas deportivas y casuales
- Conocer información detallada sobre cada producto
- Contactar fácilmente con la tienda
- Leer reseñas de otros clientes
- Disfrutar de una experiencia de navegación fluida en cualquier dispositivo
